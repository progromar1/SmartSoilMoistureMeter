# SmartSoilMoistureMeter

The plant soil moisture meter is a household device which optimizes userâ€™s watering schedules for plants. Using an Arduino Uno, Humidity Sensor, ADC Chip, LED, and enclosed in a custom 3D rendered box; the device measures moisture in the soil and outputs in the form of light what to do. Red means water the plant, Green means good as is, and Blue means the plant is overwatered.
